01. Get To Know Me

About Me

I am a backend-focused developer specializing in building robust, secure, and scalable server-side applications with Java and Spring Boot. My work centers on designing clean RESTful APIs, implementing authentication and authorization layers, and writing business logic that solves real-world problems.

I have hands-on experience with relational databases including MySQL and PostgreSQL, and I follow layered architecture patterns to keep codebases maintainable and testable. On the mobile side, I develop Android applications using modern patterns like MVVM with Retrofit for seamless API integration.

I value code clarity over cleverness, thorough documentation, and building systems that other developers can confidently extend. Every project I take on is treated as production-grade work.
